A custom designed robtic arm using a simple 3 bar linkage with only one actuation point to control the tip of the longest bar in the linkage.
Components include a Geared DC Motor, SG90 Servo Motor, and NodeMCU ESP8266
Used Arduino IDE to code the entire thing
The ESP8266 brodcasts a web server that can be used to control the individual positions of the DC motor and and the servo motor.
